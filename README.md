# Лабиринт сокровищ

**Лабиринт сокровищ** — это текстовая приключенческая игра на Python, где игрок исследует комнаты, решает загадки, собирает предметы и ищет сокровище.

## Описание игры

- Игрок начинает в темном входе лабиринта.
- В каждой комнате могут быть предметы, загадки и ловушки.
- Цель: найти ключ и открыть сундук с сокровищами в **treasure_room**.
- Игрок может взаимодействовать с предметами, решать загадки и перемещаться между комнатами.

## Установка
```bush
make install
```
## Запуск игры
```bush
make project
```
## Вариант прохождения игры
```text
Добро пожаловать в Лабиринт сокровищ!
== ENTRANCE ==
Вы в темном входе лабиринта. Стены покрыты мхом. На полу лежит старый факел.
Заметные предметы: torch
Выходы: north, east
> take torch
Вы подняли: torch
> go east
Вы переместились в trap_room.
Комната с хитрой плиточной поломкой. На стене видна надпись: "Осторожно — ловушка".
Заметные предметы: rusty_key
Выходы: west, south
> solve
Система плит активна. Чтобы пройти, назовите слово "шаг" три раза подряд (введите "шаг шаг шаг")
> шаг шаг шаг
Загадка решена!
> take rusty_key 
Вы подняли: rusty_key
> go west
Вы переместились в entrance.
Вы в темном входе лабиринта. Стены покрыты мхом. На полу лежит старый факел.
Выходы: north, east
Вы слышите странный шорох где-то рядом...
> go north
Вы переместились в hall.
Большой зал с эхом. По центру стоит пьедестал с запечатанным сундуком.
Выходы: south, west, north
> go north
Вы переместились в treasure_room.
Комната, на столе большой сундук. Дверь заперта — нужен особый ключ.
Заметные предметы: treasure_chest
Выходы: south
> use rusty_key
Вы применяете ключ, и замок щёлкает. Сундук открыт!
В сундуке сокровище! Вы победили!

```
